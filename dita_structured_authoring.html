DITA Structured Authoring

This course provides a comprehensive introduction to DITA (Darwin Information Typing Architecture) structured authoring.

You will learn the core principles, key components, and practical applications of DITA to create consistent, reusable, and scalable technical documentation.

Course Audience

This course is designed for junior to mid-level technical writers, content creators, and content professionals who are looking to understand and apply structured authoring principles using DITA.

Course Structure

  • Part I: Introduction to DITA
  • Part II: DITA Maps
  • Part III: Topic Types
  • Part IV: Short Descriptions
  • Part V: Content Reuse
  • Part VI: Publishing DITA Content
  • Conclusion
  • Part I: Introduction to DITA
    This module introduces the fundamental concepts and benefits of DITA structured authoring.
  • Part II: DITA Maps
    This module introduces DITA maps and their role in organizing and linking topics for publication.
  • Part III: Topic Types
    This module introduces core DITA topic types and their appropriate usage to improve content clarity and organization.
  • Part IV: Short Descriptions
    This module explores the purpose and best practices for writing effective short descriptions in DITA topics.
  • Part V: Content Reuse
    Learn about the importance, methods, and best practices for content reuse in DITA.
  • Part VI: Publishing DITA Content
    This module introduces the fundamentals of publishing DITA content across multiple output formats. It also covers translation workflows and customization options for publishing.
  • Part VII: Conclusion
    Summary of what you have learned in the DITA structured authoring course.
